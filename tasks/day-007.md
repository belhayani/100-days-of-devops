# Day 007 ‚Äî Linux SSH Authentication


---

## Scenario

The system admins team of `xFusionCorp Industries` has set up some scripts on `jump host` that run on regular intervals and perform operations on all app servers in `Stratos Datacenter`. To make these scripts work properly we need to make sure the `thor` user on jump host has password-less SSH access to all app servers through their respective sudo users (i.e `tony` for app server 1). Based on the requirements, perform the following:

Set up a password-less authentication from user `thor` on jump host to all app servers through their respective sudo users.

## Objective

Enable passwordless SSH authentication from `thor@jump_host` to:
- `tony@stapp01`
- `steve@stapp02`
- `banner@stapp03`

---

## Solution

### Step 1 ‚Äî Connect to the jump host as `thor`

Ensure you're logged in as the `thor` user on the jump host before starting.

```bash
whoami      # expected output: thor
hostname    # expected output: jump_host
```

### Step 2 ‚Äî Generate an SSH key pair

Generate an RSA key pair for `thor`. Press **Enter** at every prompt to accept defaults and skip the passphrase (required for password-less automation).

```bash
ssh-keygen -t rsa
```

This creates two files in `~/.ssh/`:
- `id_rsa` ‚Üí private key (keep secret)
- `id_rsa.pub` ‚Üí public key (shared with remote servers)

### Step 3 ‚Äî Copy the public key to each app server

Use `ssh-copy-id` to install the public key into each target user's `~/.ssh/authorized_keys`.

**App Server 1 ‚Äî `tony@stapp01`:**
```bash
ssh-copy-id tony@stapp01
# Password: Ir0nM@n
```

> A prompte with `Are you sure you want to continue connecting (yes/no)?`, type **yes**.

**App Server 2 ‚Äî `steve@stapp02`:**
```bash
ssh-copy-id steve@stapp02
# Password: Am3ric@
```
> A prompte with `Are you sure you want to continue connecting (yes/no)?`, type **yes**.

**App Server 3 ‚Äî `banner@stapp03`:**
```bash
ssh-copy-id banner@stapp03
# Password: BigGr33n
```
> A prompte with `Are you sure you want to continue connecting (yes/no)?`, type **yes**.


### Step 4 ‚Äî Verify passwordless SSH access

Test each connection ‚Äî it should return the remote hostname immediately without asking for a password.

```bash
ssh tony@stapp01 hostname      # expected: stapp01
ssh steve@stapp02 hostname     # expected: stapp02
ssh banner@stapp03 hostname    # expected: stapp03
```

If all three return the hostname cleanly ‚Üí **task completed successfully.** ‚úÖ

## About This Repository

This repository documents my solutions to 100 practical DevOps challenges covering Linux administration, Git, Docker, Kubernetes, Jenkins, Ansible, and Terraform. Each task simulates a real production scenario that a DevOps engineer might face on the job.

**How it’s organized:** One Markdown file per day under `tasks/day-XXX.md`, each containing the scenario and its full solution.
